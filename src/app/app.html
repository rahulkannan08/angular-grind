Description of Files and Folders

src - Contains source code and resources.

app - Contains Angular components, templates, and application logic.

index.html - Loads the Angular application in the browser.

main.ts - Entry point of the Angular application.

package.json - Contains project dependencies and npm scripts.

angular.json - Stores Angular workspace and build configuration.
